# 🦅 Antigravity Intelligence Agency (A.I.A.) - 行动指南

> **"Don't Trade, Build Tools." (不炒作，只造铲子)**

欢迎来到您的私人情报局。本系统由三大司局组成，旨在从全网噪音中提取 **现金流 (Cash Flow)** 和 **资产 (Assets)** 机会。

---

## 🗺️ 1. 组织架构 (The Architecture)

您的情报局由三个独立的“部门”组成，分别负责不同的战场：

| 部门 (Dept) | 代码 (Code) | 职责 (Mission) | 核心逻辑 (Logic) |
| :--- | :--- | :--- | :--- |
| **🔭 战略司** | `run_mission.py` | **看宏观** (Trends) | 扫描 Product Hunt / Hacker News，结合 **Grok** 识别市场风向。 |
| **💰 战术司** | `run_bounty_hunter.py` | **找现金** (Revenue) | 扫描 V2EX / 小红书 / Chrome，寻找 **"急需解决的痛苦"**。 |
| **⛏️ 工具司** | `run_alpha_radar.py` | **捡金矿** (Web3) | 监听 Solana + AI Agent，寻找 **"丑陋但赚钱的代码"** 进行包装。 |

---

## 🧠 2. 核心原理 (Core Principles)

本系统不仅仅是爬虫，它植入了您的**商业价值观**：

### A. 绝望指数 (Desperation Score) 🌡️
*   **应用**: V2EX, 小红书
*   **原理**: 我们不关心“技术问题”，我们只关心“情绪问题”。
*   **算法**: 如果帖子包含关键词 `救命`、`有偿`、`急`、`崩溃`、`红包`，系统会给予 **+100分** 的权重。
*   **目的**: **优先展示愿意直接掏钱的客户**，而不是只会白嫖的咨询者。

### B. 丑陋现金牛 (Ugly Cash Cows) 🐮
*   **应用**: Chrome Web Store
*   **原理**: 最好的生意是“很多人在用，但都在骂很难用”的产品。
*   **标准**: `用户数 > 5000` 且 `评分 < 3.8`。
*   **机会**: 它的 **1星差评 (Kill Shot)** 就是您的 **产品需求文档 (PRD)**。照着改，就能抢走它的用户。

### C. 卖铲子策略 (Shovel Seller) ⛏️
*   **应用**: Web3 (Solana)
*   **原理**: 哪怕是在淘金热中，挖金矿也是有风险的，但卖铲子永远稳赚。
*   **标准**: 系统自动过滤掉 `Airdrop` (空投/发币) 等噪音，只寻找 `GitHub` 上的 **开源工具 (Bots/Scripts)**。
*   **包装潜力 (Wrapper Potential)**: 越是只有命令行 (CLI) 的难用代码，评分越高 —— 因为您可以给它套一个 GUI 界面卖给普通人。

## 🕹️ 3. 使用方法 (How to Run)

每天早上，请在终端 (Terminal) 依次执行以下三条指令：

### 第 1 步：看宏观
```powershell
python D:\Intel_Briefing\run_mission.py
```
> **产出**: `reports/daily_briefings/Morning_Report...`
> **看点**: 今天 Product Hunt 第一名是谁？Grok 觉得它不仅好在哪里，还差在哪里？

### 第 2 步：找现金
```powershell
python D:\Intel_Briefing\run_bounty_hunter.py
```
> **产出**: `reports/tactical/Hit_List...`
> **看点**: 
> *   **V2EX**: 有没有带 🔥 图标的紧急外包？
> *   **小红书**: 点击生成的“手动搜索链接”，看看学生和打工人在为什么发愁？
> *   **Chrome**: 有没有新的“丑陋现金牛”出现？

### 第 3 步：捡金矿
```powershell
python D:\Intel_Briefing\run_alpha_radar.py
```
> **产出**: `reports/web3/Alpha_Leak...`
> **看点**: 昨晚有没有新的 Solana 抢币脚本开源？能不能包装一下？
